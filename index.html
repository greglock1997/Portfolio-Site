Full Stack Language Learning App

Description

A full stack application that allows users to login to their own account learn Japanese characters and then practice them in a quiz-like game. Upon opening the application the user will be asked to login before learning, if the user lacks an account by registering using simply a username and password, no email address required. After logging in with a new account the user will be shown various flashcards of Japanese characters inlcuding their pronunciation and their english translation. After learning a few new characters the user will be presented with a short quiz to test their knowledge.

Each user has their scores tracked for each vocabulary item and saved to a database so that in future they will be quizzed on each character based on their current level of knowledge, a basic version of a spaced-repition-system. Additionally, if the user logs in as an admin (username: Admin, password: Password), they will get access to the edit and create pages. These extra pages allow the user to add new vocabulary items to the database as well as delete or edit existing items.
